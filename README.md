# terminator
автоматизированная торговая система для работы с акциями, построенная на принципах объектно-ориентированного проектирования. Архитектура приложения включает строгую валидацию входящих рыночных данных и конфигураций с помощью Pydantic, надёжное хранение и управление состоянием через SQLAlchemy
